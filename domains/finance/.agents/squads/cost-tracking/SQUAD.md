# Squad: Cost Tracking

## Mission
Monitor and optimize AI/infrastructure spending.

## Agents

| Agent | Role | Trigger | Output |
|-------|------|---------|--------|
| cost-monitor | Track daily spend | Daily | alerts |
| budget-analyzer | Analyze budget vs actual | Weekly | reports |
| optimizer | Find cost reduction opportunities | Monthly | recommendations |

## Data Sources
- Langfuse (LLM costs)
- Cloud billing APIs
- Infrastructure metrics

## Success Metrics
- Budget adherence
- Cost per execution
- Month-over-month trends
