# Engineering Domain

## Mission
Build and maintain reliable systems and infrastructure.

## Squads

| Squad | Purpose |
|-------|---------|
| platform | Infrastructure, DevOps, CI/CD |
| backend | APIs, services, databases |
| frontend | UI, UX, web applications |

## Standards

- Code review required for all PRs
- Tests for new features
- Documentation for APIs

## Outputs
- Code in `src/`
- Docs in `docs/`
- Issues in GitHub
